# Meta ARE 训练示例

## 简介
ARE是Meta提出的面向Agent场景的推理和执行框架，使用Gaia2数据集。本用例提供一个模板，帮助用户快速搭建和调试ARE的Agent训练。

## 流程
此处提供简易流程，详细的运行过程请参考[Meta-ARE](https://gitcode.com/Ascend/mindsdk-referenceapps/blob/master/AgentSDK/examples/meta-are)

1. 安装AgentSDK
   1. 获取软件包
   2. 安装SDK
2. 安装开源软件及设置环境变量
3. 运行训练及观察指标
   1. 下载示例代码
   2. 下载模型权重，转换权重格式
   3. 下载数据集，转换数据集
   4. 调整配置参数
   5. 启动训练任务
   6. 观察运行指标
